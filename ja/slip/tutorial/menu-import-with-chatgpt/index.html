AIで素早くメニュー取り込み（約3分）

ChatGPT でメニュー画像を JSON に変換し、Slip に取り込みます。

Slip はメニュー画像を自動で読み取りません。
このガイドでは ChatGPT を使って Slip 用 JSON を作成します。

こんな方におすすめ:

  • すでに印刷または画像のメニューがある
  • メニューデータを素早く作りたい

手順

  1. ステップ1 — ChatGPT を開く

    スマホまたはPCで ChatGPT を開きます。

  2. ステップ2 — メニュー画像をアップロード

    • 印刷されたメニューを撮影
    • または既存の画像をアップロード
    • 読み取れる程度でOK — 完璧なトリミングは不要

  3. ステップ3 — プロンプトをコピー＆ペースト

    下のプロンプトをコピーして ChatGPT に貼り付けます。

    あなたはレストランメニューのOCRと構造化の専門家です。ユーザーがアップロードしたメニュー画像を読み取り、カテゴリ、商品、価格、追加、オプションを抽出し、下記のJSON構造に合う結果を出力してください（出力はJSONのみ。余計なテキストやMarkdownは不要）：

JSON structure:
{
  "sections": [
    {
      "name": "Category name",
      "sortOrder": 1,
      "menuItems": [
        {
          "name": "Item name",
          "price": 0,
          "isAvailable": true,
          "sortOrder": 1,
          "isAddon": false,
          "productUUID": null
        }
      ]
    }
  ],
  "modifiers": [
    { "term": "Modifier", "enabled": true }
  ]
}

Rules:

1. sortOrder は画像内の順番で、1から始める。
2. availability が明記されていない場合は isAvailable を true にする。
3. 商品が追加料金の項目（例: "Add pearls", "Add cheese"）なら isAddon を true、そうでなければ false。
4. productUUID は null（または省略してもよいが一貫性を保つ）。
5. カテゴリがない場合は "Uncategorized" の1カテゴリを作成する。
6. 価格は通貨記号なしの数値（小数可）。
7. modifiers は画像内の一般的な選択肢（例: "Less ice", "Less sugar", "No spice", "No ice"）。なければ空配列。
8. 同じ商品が複数カテゴリにある場合はカテゴリごとに1件ずつ残し、統合しない。
9. 出力はダブルクオートを使った有効なJSONにする。

画像を解析してJSONを出力してください。

  4. ステップ4 — menu.json を取得

    ChatGPT の回答が返ってきたら、JSON として有効か確認し、{ から } までをすべてコピーします。

    JSON 出力プレビュー

    {
  "sections": [
    {
      "name": "ドリンク",
      "sortOrder": 1,
      "menuItems": [
        {
          "name": "紅茶",
          "price": 30,
          "isAvailable": true,
          "sortOrder": 1,
          "isAddon": false,
          "productUUID": null
        },
        {
          "name": "タピオカ追加",
          "price": 10,
          "isAvailable": true,
          "sortOrder": 2,
          "isAddon": true,
          "productUUID": null
        }
      ]
    }
  ],
  "modifiers": [
    { "term": "氷少なめ", "enabled": true },
    { "term": "甘さ控えめ", "enabled": true }
  ]
}

  5. ステップ5 — Slip に取り込み

    Slip に戻る → メニュー取り込み → JSON を貼り付け → プレビュー → 取り込み。

重要な注意

  • この内容は AI 生成のため、手動での確認が必要な場合があります
  • Slip は取り込み前にプレビューを表示します
  • 価格とカテゴリを確認してください

免責事項

このガイドは運用の参考情報です。ChatGPT の出力確認はユーザーの責任で行ってください。
